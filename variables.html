Variables

Definition

  • One of the most powerful features of a programming language is the ability to manipulate variables. A variable is a name that refers to a value.
  • Some of u may be like why should one refer a value by its name , and that's because even when the value is a very long paragraph it can be refered simply by its varaiable name.
  • Also consider during the program if the value may obtain some changes , so if we refer a value wih a variable name ,we can refer it to another value if needed during the program.

Variable names

  • A variable name may start with an '_' or alphabet followed by numbers or alphabets or '_'.
  • A variabe name should not contain python key words which are the words having special meaning in python.
  • valid variable names eg. _key, box1.
  • Invalid ones eg. 1g, a b .
  • As we have seen variable names rules now click datatypes to see the types of values that a variable can refer.

DATA TYPES

Definition

  • data types means the different types of values that python supports which variables can refer.
  • lets see some of the basic ones 1.integer, 2.float, 3.strings, 4.boolean. The rest of the data types are covered as we move on.

Integer

  • Integers are numbers which can have signed values without decimal places.
  • eg. 1, -5

Float

  • Float are signed numbers that can have fractional values
  • eg. -1.5 , 2.2

Strings

  • Strings are the ones which can have one or more characters which includes numbers , alphabets and special symbols.
  • Strings should be enclosed with single quotes or double quotes . the recommended one is double qotes.
  • eg. "abc", "a1b", "1", "@1"
  • Eventhough if a partcular string contains only numers . it cannot be used in mathematical operations.

Boolean

  • boolean type has only two values.
  • The two boolean values are True and False
  • These values are note enclosed in quotes and are case sensitive.
  • boolean is one of the important types in any language. we will see its uses soon.

Assignment Statement

  • An assignment statement assigns the value in the right hand side of the '=' to the variable in the left hand side of '='.
  • assigning a integer to a variable. eg : x = 1 . here x is the variable name and 1 is integer.
  • assigning a string to a variable. eg : x = "hello".
  • assigning a float to a variable. eg : x = 1.5.
  • assigning a boolean to a variable. eg : x = True.
