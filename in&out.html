print()

  • Often while we are programming we must want to display something in the screen as an output.
  • this is done with the help of print().
  • printing an integer or float is simple, just enter the number between the paranthesis. eg: print(5) . this displays the number 5.
  • printing an string means as usual u should enclose the string in quotes. eg: print("india") .this would display india.
  • printing a variable means u can just enter the name of the variable between paranthesis.if variable name is x , then print(x). this would print the value of x.
  • printing a group of variables,values means u should print them by seperating them by ',', enclosed by paranthesis. note that comma will not be printed.

Examples

  • print(5.5) . Output is 5.5
  • print(True) . Output is True. not here true is python keyword which has boolean value True.
  • x = 6
print(x)
print(x,5,5.5)

    5
6 5 5.5

type()

  • type() gives us the type of the value or variable entered between paranthesis.
  • eg. type(5) returns <class 'int'>
  • this is really an important function to know the type of the variable.

INPUT

input()

  • input() can display a number or a string on the screen and get an input from the user as string type.
  • for eg: 
    x = input("Enter a number  :")
print(x)

output of the code will be

Enter a number  :1
1
  • even though u might think as u have entered a number, input() gives it to us as a string. u can use type() to check it. so it has to be converted.

type conversion

Definition

  • since the input returns only strings we can convert it to other by using type conversion.
  • for eg. if you want to convert float 5.5 to int u may use int(5.5). resulting in 5 as output
  • similarly float(), string() can be used.

Example

    x = input('Enter a number : ')
print(x,type(x))
x = float(x)
print(x,type(x))
x = int(x)
print(x,type(x))

output of the code will be , if we enter the number 2.2

Enter a number : 2.2
<class 'str'>

2 <class 'int'>
