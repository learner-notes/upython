OPERATORS

  • Operators are symbols that performs special operations in the language.
  • The basic types of operators are assignment('=', which is used to assign a value to a variable), arithmetic, relational, logical operators.

Arithmetic operators

  • Arithmetic operators operates on numbers used for addition, subraction, multiplication, division, power.
  • Arithmetic operator returns a number
  • addition - + - eg: 5+6 results 11
  • subraction - - - eg: 5-6 results -1
  • multiplication - * - eg: 5*6 results 30
  • division - / - eg: 6/6 results 1
  • power - ** - eg: 6**2 results 36
  • modulus - % - eg: 6%1 results 1 gives remainder of the division 1

Operands

  • operands are the entities on which operator acts.
  • eg: 5*6 here * is the operator 5,6 operands.

Order of precedence

  • The order of evaluation of the operators, depends on the rules of precedence,
    paranthesis (),
    exponentation **,
    multiplication and division *,\,
    addition and subraction +,-,
    relational operators,
    logical operators,
    assignment operator.
  • operators with same precedence are evaluated from left to right.
  • When in doubt, always put parentheses in your expressions to make sure the computations are performed in the order you intend.

Relational operators or boolean

  • relational operators are used to compare two values of similar types, (mostly) and returns either true or false based on the result of the comparison.
  • lets assume two variables x,y with some values strings or numbers.
  • == - compare equality - eg: x==y,
    != - not equal - eg: x!=y,
    > - greater - eg: x>y ,
    < - lesser - eg: x<y,
    >= - greater than or equal - eg: x>=y,
    <= - lesser than or equal - eg: x<=y,
    is - the same - eg: x is y,
    is not - not the same - eg: x is not y,
  • Although these operations are probably familiar to you, the Python symbols are different from the mathematical symbols for the same operations. A common error is to use a single equal sign (=) instead of a double equal sign (==). Remember that = is an assignment operator and == is a comparison operator.
  • Examples:
    print( 5==5 )
    print( 8<2 )
    print( 'a'!='a' )
    print( '1' == 1 )
    print( 1 > '1' )

    the output of the code will be,

    True
    False
    False
    False
    Traceback (most recent call last):
    File "<string>", line 5, in <module>
    TypeError: '>' not supported between instances of 'int' and 'str'

Logical operators

  • Logical operators are used to combine the result of the relational expressions.
  • and - returns true if and only if all the inputs are true.
  • or - returns true if one of the inputs is true
  • not - inverses true or false
  • Example:
    print( 5==1 and 5==5 )
    print( 5==1 or 5==5 )
    print( not 5==1 and 5==5 )

    the output of the code :

    False
    True
    True
